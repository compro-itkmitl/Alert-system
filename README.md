# Install & Setup Website Alert-system
1. download หรือ clone project
2. cd front-end และ ต่อด้วย yarn install , yarn buld

# Run
yarn dev 
ตัวเว็บไซด์จะไปที่ localhost:3000

ถ้าเข้าได้แล้วจะเจอหน้าตาเว็บไซด์แบบนี้
